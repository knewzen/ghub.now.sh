Welcome to GHUB NOW

A micro-service that redirects to the repository page of given package name.

Usage

https://ghub.now.sh/<package-name>

Example

Try some of these: ghub-now, hela, preact, gatsby, assemble.

Why?

Because GHUB.IO is slow, inconsistent and fells down too much.
